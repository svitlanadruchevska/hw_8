1. Создаем репозиторий на github.com  c назанием react-lesson-1
    - https://github.com/new

2. На странице репозитория вы увидите готовые команды для
   подключения созданного репозитория на Гитхабе к уже существующему
   репозиторию у вас на компьютере.

    Выполните эти шаги:

    git remote add origin https://github.com/<Ваше имя на github>/react-lesson-1.git
    git branch -M main
    git push -u origin main

3. Обновляем стрицу на github и проверяем что все но месте


Теперь не важно, какие изменения делаются в локальном репозитории,
на Github все коммиты попадут только после команды git push.
